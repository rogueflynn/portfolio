About Me

My name is Victor Gonzalz and I am a programmer/developer. I'm a Computer
Science major. Most of my interests has been in web developing
and database connectivity. My other interests include game development and desktop
applicaton development.

A lot of my experience has been working with object oriented languages such as Java and C#.
I have used these languages to create games and integrate with SQL based databases.
Recently, I began exploring working with Node.js, express and NoSQL databases like MongoDB.

I believe that programming is about problem solving and using programming languages
to solve problems. I work well on team and always willing to keep learning new things
My work ethic is strong and I would be a valuable asset to any company.
